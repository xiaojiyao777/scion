Failure:

```text
scion/tests/test_campaign.py::TestPromoteWeightOptimizationHook::test_on_promote_runs_weight_optimization
```

Classification:

This is a pre-existing compatibility-test mismatch, not a Phase 1 regression.
The test expects the legacy `_on_promote()` path to call
`CampaignManager._run_weight_optimization()` directly. Current production code
routes promotion-time weight optimization through
`AsyncWeightOptCoordinator.spawn_for_promoted_champion()` /
`run_for_promoted_champion_sync()` during `_commit_promote_plan()`.

Phase 1 integration should either update this legacy test to assert the current
coordinator behavior or keep it explicitly marked as a known baseline failure
until promotion extraction is complete.
