## 5. 关键设计决策

1. **Parallel paths** — adapter/metric_specs 为可选参数，None 时走 legacy 路径，保证后向兼容
2. **Persist facts, rebuild views** — family stats 和 failure summary 从 lineage 查询派生，不维护独立 mutable state
3. **MILP offline** — 预计算结果存为 `milp_bounds/<stem>.json`，adapter.estimate_lower_bound 读取，report-only
4. **Immutable snapshots** — weight-opt 创建新目录 `champion_v{N}_r{R}`，原始 snapshot 不可变
5. **Stage-aware stale** — weight 更新只影响 screening/explore，validation/frozen 不中断
6. **Scoring decouple** — weight-opt 用 decisive metric delta，不依赖 SCION_SPLITS_WEIGHT
